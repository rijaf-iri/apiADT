Fill in this form to create an account.

  

User Access Levels

  • Administrator :  super user, able to create account
  • All Stations :  access to all stations
  • Selected Stations :  access to a selected stations

User Action

  • Display and Download :  able to download data and display observations in a table, charts and maps
  • Display Only :  display observation in a charts and maps

Stations Selection

  • Geographic :  select stations from a geographical support (polygons)
  • Stations :  select stations from a list
